1. null এবং undefined এর মধ্যে পার্থক্য:

 null হলো এক ধরনের মান যা ইচ্ছাকৃতভাবে শূন্য বা খালি বোঝাতে ব্যবহার করা হয়।

 undefined হলো এমন একটি অবস্থা যখন ভেরিয়েবল ডিক্লেয়ার করা হয়েছে কিন্তু কোনো মান দেওয়া হয়নি।

2. map()  এবং forEach() এর  পার্থক্য:

 map() কোনো array এর প্রতিটি উপাদানকে প্রসেস করে নতুন array রিটার্ন করে।

 forEach() শুধুমাত্র array এর প্রতিটি উপাদানকে প্রসেস করে কিন্তু কিছু রিটার্ন করে না।

3. == এবং === এর পার্থক্য:

 == মান তুলনা করে, টাইপ কনভার্সন করে।

 === মান এবং টাইপ উভয়ই তুলনা করে, টাইপ কনভার্সন করে না।

4. async/await এর গুরুত্ব (API data fetching ):

 API data fetch করার সময় asynchronous কোডকে synchronous এর মতো লিখতে সাহায্য করে।

 কোড পড়া সহজ হয় এবং প্রমিস হ্যান্ডলিং সহজ হয়।

5. JavaScript এ Scope এর ধারণা :

 Global Scope: স্কোপ যা সব জায়গা থেকে অ্যাক্সেস করা যায়।

 Function Scope: ফাংশনের ভিতরে ডিফাইন করা ভেরিয়েবল শুধুমাত্র সেই ফাংশনের ভিতরে অ্যাক্সেস করা যায়।

 Block Scope:  ব্লকের ভিতরে ডিফাইন করা ভেরিয়েবল শুধু ওই ব্লকের ভিতরে অ্যাক্সেস করা যায়।
